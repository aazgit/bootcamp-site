Curriculum & Syllabus

A comprehensive 12-week journey through Figma mastery, UI/UX design principles, and industry-ready skills with hands-on projects and real-world applications.

Download Complete Syllabus PDF

Program Structure

Phase 1
Foundations
Weeks 1-4

Master Figma basics, design principles, and establish your workflow.

Phase 2
Application
Weeks 5-8

Apply skills to real projects, learn advanced techniques and user research.

Phase 3
Mastery
Weeks 9-12

Build portfolio, capstone project, and prepare for career transition.

Phase 1: Foundations (Weeks 1-4)

01

Week 1

Introduction to Figma & Design Thinking

Topics:

  • Figma interface and navigation
  • Design thinking methodology
  • Basic shapes, text, and layers
  • Introduction to design systems

Deliverables:

  • Personal workspace setup
  • First design exercise
  • Design thinking framework assignment

Live Sessions: Figma basics workshop, Design thinking lecture

02

Week 2

UI Design Principles & Typography

Topics:

  • Visual hierarchy and composition
  • Typography fundamentals
  • Color theory and application
  • Grid systems and layout

Deliverables:

  • Typography poster design
  • Color palette creation
  • Grid-based layout exercise

Live Sessions: Typography workshop, UI principles critique

03

Week 3

Components & Auto Layout

Topics:

  • Creating and managing components
  • Variants and properties
  • Auto Layout fundamentals
  • Responsive design principles

Deliverables:

  • Component library creation
  • Responsive card design
  • Auto Layout practice exercises

Live Sessions: Components masterclass, Auto Layout workshop

04

Week 4

Basic Prototyping & Interactions

Topics:

  • Figma prototyping features
  • Basic interactions and transitions
  • User flow mapping
  • First mobile app design

Deliverables:

  • Project 1: Mobile app landing screens
  • Interactive prototype
  • User flow documentation

Live Sessions: Prototyping tutorial, Project 1 critique

Phase 2: Application (Weeks 5-8)

05

Week 5

User Research & Personas

Topics:

  • User research methodologies
  • Creating user personas
  • Journey mapping
  • Competitive analysis techniques

Deliverables:

  • User research plan
  • Persona development
  • Competitive analysis report

Live Sessions: Research methods workshop, Persona creation lab

06

Week 6

Advanced Figma Techniques

Topics:

  • Advanced component architecture
  • Boolean operations and masks
  • Plugins and workflow optimization
  • Team collaboration features

Deliverables:

  • Advanced component system
  • Icon library creation
  • Workflow optimization plan

Live Sessions: Advanced techniques masterclass, Plugin showcase

07

Week 7

Design Systems & Scalability

Topics:

  • Design system architecture
  • Atomic design methodology
  • Documentation and governance
  • Cross-platform consistency

Deliverables:

  • Complete design system
  • Style guide documentation
  • Component usage guidelines

Live Sessions: Design systems workshop, Documentation best practices

08

Week 8

Usability Testing & Iteration

Topics:

  • Usability testing methods
  • Testing tool integration
  • Analyzing user feedback
  • Design iteration processes

Deliverables:

  • Project 2: E-commerce dashboard
  • Usability test plan and results
  • Iteration documentation

Live Sessions: Testing methods workshop, Project 2 critique

Phase 3: Mastery (Weeks 9-12)

09

Week 9

Advanced Prototyping & Micro-interactions

Topics:

  • Smart animate and advanced transitions
  • Micro-interaction design
  • Animation principles in UI
  • Prototype testing and sharing

Deliverables:

  • Interactive app prototype
  • Micro-interaction library
  • Animation showcase

Live Sessions: Advanced prototyping lab, Animation workshop

Week 10

Developer Handoff & Collaboration

Topics:

  • Design specifications and handoff
  • Developer collaboration best practices
  • Design QA and review processes
  • Version control for designers

Deliverables:

  • Developer handoff documentation
  • Design specifications guide
  • Collaboration workflow plan

Live Sessions: Handoff workshop, Developer collaboration panel

Week 11

Portfolio Development & Presentation

Topics:

  • Portfolio strategy and curation
  • Case study development
  • Presentation skills for designers

Deliverables:

  • Portfolio website wireframes
  • Case study drafts
  • Personal brand strategy

Live Sessions: Portfolio review, Presentation skills workshop

Week 12

Capstone Project & Career Preparation

Topics:

  • Capstone project execution
  • Career transition strategies
  • Job search and interview preparation

Deliverables:

  • Complete portfolio presentation
  • Career roadmap and goals
